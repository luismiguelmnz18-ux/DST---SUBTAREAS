🔄 Sincronizar datos
Conecta vía Google Apps Script (puente gratuito). Pega la URL del script implementado:
📋 Instrucciones: Google Apps Script (clic para expandir)
1. Ve a script.google.com → Nuevo proyecto
2. Borra el código inicial y pega este:
const JIRA_URL   = "TU_URL_JIRA";
const JIRA_EMAIL = "TU_EMAIL";
const JIRA_TOKEN = "TU_API_TOKEN";
const PROJECTS   = ["CORE","CRM","BI"];

function doGet(e) {
  const jql = "project in (" + PROJECTS.map(p => JSON.stringify(p)).join(",") + ") AND issuetype in (Sub-task) AND summary ~ "DST" ORDER BY duedate ASC";
  const url = JIRA_URL + "/rest/api/3/search?jql=" + encodeURIComponent(jql) + "&maxResults=500&fields=summary,assignee,project,status,issuetype,duedate,resolutiondate,created,parent,timeoriginalestimate,timespent";
  const resp = UrlFetchApp.fetch(url, { headers: { Authorization: "Basic " + Utilities.base64Encode(JIRA_EMAIL + ":" + JIRA_TOKEN) }, muteHttpExceptions: true });
  const data = JSON.parse(resp.getContentText());
  return ContentService.createTextOutput(JSON.stringify(data)).setMimeType(ContentService.MimeType.JSON);
}
3. Reemplaza TU_URL_JIRA, TU_EMAIL y TU_API_TOKEN con tus datos reales
4. Clic Implementar → Nueva implementación → Aplicación web
5. "Ejecutar como": Yo · "Quién tiene acceso": Cualquier persona
6. Copia la URL generada y pégala arriba en el campo de sincronización
